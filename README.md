Тестовое задание для допуска к онлайн-стажировке от JavaRush

Cтандартное CRUD приложение, которое отображает список всех юзеров в базе (с пейджингом). С возможностью их удаления, редактирования, добавления новых. 

Запрос для создания базы данных, таблицы и заполнения ее тестовыми данными осуществляется с помощью sql запроса из файла test_user.sql

После деплоя в Томкэт приложение будет доступно по URL http://localhost:8080/index.jsp

Используемые технологии: Maven, Spring, Hibernate, Tomcat, Mysql.
